# Personal Blog

My personal blog and portfolio built with Astro and AstroPaper theme.
You are welcome to clone and use this project for your own blog.

## How to run locally

1. Clone the repository
```bash
git clone https://github.com/alanJames00/personal-blog
```

2. Install dependencies
```bash
cd personal-blog
pnpm install
```

3. Run the development server
```bash
pnpm run dev
```

4. Make changes to the content and take it from here.
